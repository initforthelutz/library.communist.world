    The following collection of Essays in Self-Criticism is available in both a single file containing the entire text (553k) or in two smaller files. An index from the text is also included, although the subject headings and page references in the index are NOT linked to the various sections of the text. Still, it should prove useful once you have downloaded and saved all sections of the book (in one folder) on your hard drive. You will need to use two browsers (or two windows of the same browser) at the same time: one for viewing the text, and the other for viewing the index. Using the browser's "Find..." command, key words from the index can be entered, or, you can go to specific page references by entering "page x" (where "x" is a number -- there must be a space between the word "page" and the number).
